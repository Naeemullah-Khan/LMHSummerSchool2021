Course Instructions

Save all your work in a single Colab notebook for each day.

The notebooks for each day should be stored in a single folder which will be sent to the instructor as a zip folder at the of the course.

The assignment for each day will be used as the attendance for that day.

Assignments should be completed individually, however, discussions amongst students is allowed and encouraged.
